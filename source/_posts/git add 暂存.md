# add命令

## git add 命令可将该文件添加到暂存区。

添加一个或多个文件到暂存区：

```bash
git add [fle1] [file2] ...
```

添加指定目录到暂存区，包括子目录：

```bash
git add [dir]
```

添加当前目录下的所有文件到暂存区：

```bash
git add .
```
