# golang 统计代码行数

## 使用gocloc工具进行统计

获取gocloc工具：

```bash
go get -u github.com/hhatto/gocloc/cmd/gocloc
```
使用gocloc命令 gocloc[目录路径或文件路径] 进行统计

```bash
gocloc .
```

`gocloc`会按类别统计路径下有多少文件，文件中的空行数，注释行数以及实际代码行数。如果目录下某类文件不需要统计，可以使用参数`--exclue-ext=[文件后缀]`

例如：

```bash
gocloc --exclue-ext=txt .
```
