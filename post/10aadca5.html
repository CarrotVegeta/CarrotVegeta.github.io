发表于2022-02-16|更新于2022-02-16|linux

命令行配置代理

export http_proxy=http://127.0.0.1:1087

export https_proxy=$http_proxy

此处的IP和端口需要根据具体你代理的实际信息填写，可查看VPN客户端的config.json

"inbounds": [
 {
 "listen": "127.0.0.1",
 "protocol": "socks",

 },

 },

 "protocol": "http",

 "timeout": 360
 },
 "port": "1087"
 }

收到200响应表示成功
